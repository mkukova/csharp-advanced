# 1.Lab: Functional Programming

1. 2.Sum Numbers

Write a program that reads a line of **integers** separated by **&quot;, &quot;**. Print on two lines the **count** of numbers and their **sum.**

### Examples

| **Input** | **Output** |
| --- | --- |
| 4, 2, 1, 3, 5, 7, 1, 4, 2, 12 | 1041 |
| 2, 4, 6 | 312 |
